This folder contains the data from the Stanford Flu Spread Study.  For
questions/comments on the data, please contact Marcel Salathe at
salathe@psu.edu

Contents:

roles.txt
  This file has user IDs and whether the subject was a
student, staff, teacher, or other

moteFiles
  This folder contains one file per receiver mote. For
example, the file node-1 will contain all the beacons that node 1
received during the experiment.
  The format of those files is as follows: 
senderID senderSequenceNumber RSSIvalue localTime globalTime

edgeLists
  See readme file in the folder.
